## 1. アカウント登録と初期設定

1. **TradePost Proのウェブサイトにアクセス**
   公式サイトの右上にある「新規登録」ボタンをクリックします。

2. **アカウント情報の入力**
   - メールアドレス
   - パスワード（8文字以上、英数字混合）
   - 名前（表示名）
   を入力し、「登録する」ボタンをクリックします。

3. **メールアドレスの確認**
   登録したメールアドレス宛に確認メールが届きます。メール内のリンクをクリックして登録を完了させてください。

4. **ダッシュボードへのログイン**
   登録したメールアドレスとパスワードでログインすると、ダッシュボードが表示されます。

## 2. プランの選択と決済

TradePost Proでは3つのプランをご用意しています。ご自身の運用スタイルに合わせてお選びください。

1. ダッシュボードの左メニューから「プラン設定」をクリックします。
2. 希望するプランの「このプランを選択」ボタンをクリックします。
3. Stripeの決済画面に移動しますので、クレジットカード情報を入力して決済を完了させます。
4. 決済が完了すると、自動的にダッシュボードに戻り、プランが適用されます。

## 3. MT4/MT5との連携設定

自動投稿を行うためには、お使いの取引プラットフォーム（MT4またはMT5）のデータをTradePost Proに送信する設定が必要です。

### MT5をご利用の場合（推奨）

MT5をご利用の場合は、Pythonスクリプトを使用して自動的にデータを取得・送信できます。

1. **Pythonのインストール**
   ご利用のPCまたはVPSにPython（3.9以上）がインストールされていることを確認してください。

2. **連携ツールのダウンロード**
   ダッシュボードの「MT4/MT5連携」ページから、「MT5連携ツール（ZIP）」をダウンロードして解凍します。

3. **APIキーの設定**
   ダッシュボードに表示されている「あなたのAPIキー」をコピーし、解凍したフォルダ内の `.env` ファイルに貼り付けます。

4. **ツールの実行**
   コマンドプロンプト（またはPowerShell）を開き、以下のコマンドを実行します。
   ```bash
   pip install -r requirements.txt
   python mt5_sender.py
   ```

5. **自動実行の設定**
   毎日自動で実行されるように、Windowsのタスクスケジューラに登録します。（設定方法は同梱の `README.txt` を参照してください）

### MT4をご利用の場合

MT4をご利用の場合は、専用のEA（Expert Advisor）を使用してデータをCSVに出力し、それを送信します。

1. **EAのダウンロードと設置**
   ダッシュボードから「MT4用EA（DailyTradeExporter.ex4）」をダウンロードします。
   MT4を開き、「ファイル」＞「データフォルダを開く」を選択し、`MQL4/Experts` フォルダにダウンロードしたファイルを入れます。

2. **EAの稼働**
   MT4を再起動し、ナビゲーターパネルから「DailyTradeExporter」を任意のチャートにドラッグ＆ドロップします。
   「自動売買を許可する」にチェックを入れて「OK」をクリックします。

3. **送信ツールの設定**
   ダッシュボードから「MT4データ送信ツール」をダウンロードし、MT5の場合と同様にAPIキーを設定してタスクスケジューラに登録します。

## 4. SNSアカウントの連携

各SNSに自動投稿するための設定を行います。ダッシュボードの左メニューから「SNS設定」をクリックしてください。

### X (Twitter) の設定
1. [X Developer Portal](https://developer.twitter.com/) にアクセスし、アプリを作成して以下の4つのキーを取得します。
   - API Key
   - API Key Secret
   - Access Token
   - Access Token Secret
2. TradePost ProのSNS設定画面で、取得したキーを入力して「保存」をクリックします。

### Instagram / Threads の設定
1. [Meta for Developers](https://developers.facebook.com/) でアプリを作成し、Instagram Graph APIまたはThreads APIの設定を行います。
2. 取得した「Access Token」と「アカウントID」をTradePost Proに入力して保存します。

### TikTok の設定
1. [TikTok for Developers](https://developers.tiktok.com/) でアプリを作成し、Content Posting APIの権限を取得します。
2. 取得した「Client Key」と「Client Secret」を入力して保存します。

### LINE オープンチャットの設定
1. [LINE Notify](https://notify-bot.line.me/) にアクセスし、ログインします。
2. 「マイページ」から「トークンを発行する」をクリックし、投稿先のグループを選択してトークンを発行します。
3. 発行されたトークンをTradePost Proに入力して保存します。

※ 各SNSの詳しいAPI取得手順については、別途「[SNS API申請ガイド](./SNS_API_GUIDE.md)」をご参照ください。

## 5. 投稿テンプレートと時間の設定

ダッシュボードの「投稿設定」から、画像のデザインや投稿時間をカスタマイズできます。

1. **画像テンプレートの選択**
   ご契約のプランに応じて、複数のデザインテンプレートからお好きなものを選択できます。
   - ダーククラシック（全プラン）
   - ネオングロー（全プラン）
   - ミニマルホワイト（全プラン）
   - ゴールドラグジュアリー（スタンダード以上）
   - グラデーションウェーブ（スタンダード以上）

2. **投稿時間の設定**
   毎日何時にSNSへ投稿するかを設定します。
   ※ 例：「07:00」に設定すると、前日の取引結果が毎朝7時に各SNSへ自動投稿されます。

3. **カスタムテキストの設定（プレミアムプランのみ）**
   投稿時のテキスト（キャプション）に含めるハッシュタグや、LINEオープンチャットへの誘導文を自由に編集できます。

## 6. よくある質問（FAQ）

**Q: 週末（土日）も投稿されますか？**
A: はい、投稿されます。ただし、FX市場が閉まっており取引がない場合は、「本日の取引はありませんでした」という内容の画像が生成・投稿されます。

**Q: 複数のMT4/MT5口座の合算結果を投稿できますか？**
A: 現在のバージョンでは1つの口座のみ対応しています。複数口座の対応は今後のアップデートで追加予定です。

**Q: プランのダウングレードは可能ですか？**
A: はい、ダッシュボードの「プラン設定」からいつでも変更可能です。ただし、次回更新日からの適用となります。

**Q: エラーで投稿されなかった場合、どうなりますか？**
A: システムが自動的に最大3回まで再試行（リトライ）を行います。それでも失敗した場合は、登録メールアドレス宛にエラー通知が届きます。

**Q: サポートに問い合わせる方法は？**
A: ダッシュボード右下の「サポート」ボタン、または `support@tradepost-pro.example.com` 宛にメールでご連絡ください。
